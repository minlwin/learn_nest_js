# Nest JS
Learning Nest JS 

Nest JS ဟာ REST API တွေကို ရေးသားရာမှာ အသုံးပြုနိုင်တဲ့ Node.js အပေါ်မှာ အခြေခံထားတဲ့ Server Side Framework တစ်ခုဖြစ်ပါတယ်။ Nest JS ဟာ Prograssive Javascript အပေါ်မှာ အခြေခံထားပြီး၊ Type Script ကိုလဲ Full Suport ပေးပါတယ်။ တဖန် Object Oriented Programming, Functional Programming, Functional Reactive Programming တို့နဲ့လဲဲ တွဲဖက် ရေးသားနိုင်မှာ ဖြစ်ပါတယ်။
